## 미션

### Part 1: 회원가입 & 로그인
- 이메일 가입 + **동네 설정** (직접 입력 또는 위치 인증)

### Part 2: 상품 등록
- **이미지(최대 3장)** + 제목 + 가격 + 설명 + 카테고리
- 본인만 수정/삭제 가능 (RLS)

### Part 3: 상품 목록 & 상세
- 최신순 목록 + **카테고리 필터** + **키워드 검색**
- 상세 페이지: 이미지 슬라이드, 정보, 작성자, **"관심" 버튼**

### Part 4: 채팅 (1:1 문의)
- 상품 상세에서 "채팅하기" → 판매자와 1:1 채팅
- Polling 으로 실시간 메시지 (주기적으로 DB 변화 체크)

### Part 5: 마이페이지
- 내가 등록한 상품 / 관심 상품 / 채팅 목록

### Part 6: 배포
- Vercel 배포 + 공유 가능한 URL

---

## 핵심 구조

```
[가입/로그인] → [상품 등록 (이미지+정보)] → [목록/검색] → [상세 → 채팅] → [마이페이지]
```

- **Auth:** 누가 등록했는지 / 누가 채팅하는지
- **DB:** 상품 + 사용자 + 채팅 메시지
- **Storage:** 상품 이미지 업로드
- **Realtime:** 실시간 채팅
